Benefits of measuring maturity

Immediate insights

Get a clear picture of your organisation's design strengths and opportunities.

Evidence-based decisions

Use data to inform your design strategy and prioritise improvements.

Track progress

Monitor your journey towards design maturity.

Benchmark performance

Compare your design maturity with other departments over time.

How it works

  1. Sign up

    Create a survey administrator account to set up and run surveys for your organisation.

  2. Create a survey period

    Create a survey period and invite people to complete the survey with an unique code for your organisation.

  3. People complete the survey

    Anyone with the code and link can complete the survey for your organisation.

  4. View the results

    View the results and analysis in your admin dashboard and track changes to maturity over time.

Roadmap

  • View and compare maturity scores across departments

    In future, you will be able to see and compare top level maturity scores across departments that have run surveys.

  • Add optional survey questions

    To capture department specific data, you will be able to add additional questions to the survey.

  • Export results

    You can view all data in the admin dashboard now, but in future you will be able to export the data in CSV format.

  • Provide an API

    If there is a need, we will provide an API to allow you to pull your organisation's data into your own systems.
